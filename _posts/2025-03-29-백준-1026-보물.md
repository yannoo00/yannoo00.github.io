## 문제 정보
- 문제 링크: [백준 1026 보물](https://www.acmicpc.net/problem/1026)
- 난이도: <span style="color:#B5C78A">실버4</span>
- 완료일: 2025년 3월 17일
- 유형: 그리디

### 내 코드

```C++
#include <iostream>
#include <string.h>
#include <vector>
#include <queue>
#include <cmath>
#include <algorithm>
#include <string>

using namespace std;

int n;
int a[50];
int b[50];

int main()
{
> cin >> n;
> for(int i=0; i<n; ++i)
> {
> cin >> a[i];
> }
> for(int i=0; i<n; ++i)
> {
> cin >> b[i];
> b[i] *= -1;
> }
> sort(a, a+n);
> sort(b, b+n);

> long long ans {0};
> for(int i=0; i<n; ++i)
> {
> ans += a[i] * -b[i];
> }

> cout << ans;
> return 0;
}

```

A 배열을 마음대로 정렬할 수 있으므로,  
그냥 B의 요소 중 큰 순서대로 A의 작은 수를 짝지으면 된다.
