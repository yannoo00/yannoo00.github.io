## 문제 정보
- 문제 링크: [백준 1068 트리](https://www.acmicpc.net/problem/1068)
- 난이도: <span style="color:#FFA500">골드5</span>
- 완료일: 2024년 2월 22일
- 유형: 트리
- 특이사항: 한 달 만에 풀이

### 내 코드

{% highlight C++ %} {% raw %}
#include <iostream	
#include <string	
#include <vector	

using namespace std;

int n, x, ans;
vector<int	 v;

void Searching(int parent)
{
	bool leaf = true;
	for(int i=0; i<n; i++)
	{
		if(v[i] == parent && i!=x)
		{
			Searching(i);
			leaf = false;
		}
	}
	if(leaf && parent!=-1)
	{
		ans++;
	}
}

int main(void)
{
	int temp;
	
	cin 		 n;
	for(int i=0; i<n; i++)
	{
		cin 		 temp;
		v.push_back(temp); //부모를 기록
	}
	cin 		 x;
	
	Searching(-1);
	
	cout << ans;
}

재귀를 이용해서 해결했다.

뿌리 노드부터 시작해서, 파라미터로 노드 번호를 넘기고 해당 번호의 노드를 부모로 갖는 노드들을 찾아 이 과정을 반복한다. 이 때 재귀가 실행되지 않으면 리프 노드다.   
삭제 대상인 노드는 아예 방문하지 않으면 된다.  
예외적으로 루트 노드가 삭제되는 경우를 따로 처리해주면 끝.
