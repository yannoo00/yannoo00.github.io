## 문제 정보
- 문제 링크: [백준 11055 가장 큰 증가하는 부분 수열](https://www.acmicpc.net/problem/11055)
- 난이도: <span style="color:#B5C78A">실버2</span>
- 완료일: 2023년 7월 29일
- 유형: DP

### 내 코드

{% highlight C++ %} {% raw %}
#include <iostream	
#include <vector	
#include <algorithm	

using namespace std;

int n;
int arr[1001];
int dp[1001];

int main(void)
{
	cin 		 n;
	
	int MAX = 0;
	
	for(int i=1; i<=n; i++)
	{
		cin 		 arr[i];
		
		for(int j=i-1; j	=0; j--)
		{
			if(arr[j]<arr[i] && dp[j]	dp[i])
			{
				dp[i] = dp[j];
			}
		}
		dp[i] += arr[i];
		MAX = max(MAX, dp[i]);		
	}
	cout << MAX;
}
