## 문제 정보
- 문제 링크: [백준 15681 트리와 쿼리](https://www.acmicpc.net/problem/15681)
- 난이도: <span style="color:#FFA500">골드5</span>
- 완료일: 2023년 11월 13일
- 유형: DP

### 내 코드

```C++
#include <iostream>
#include <vector>
#include <algorithm>
#include <string>
#include <vector>
#include <queue>

using namespace std;

vector<int> tree[100001];
bool visited[100001];
int children[100001];
int n, r, q, v1, v2, u;

int DFS(int node)
{
	visited[node] = true;
	int next;
	int sum = 0;
	for(int i=0; i<tree[node].size(); i++)
	{
		next = tree[node][i];
		if(visited[next] == false)
		{
			sum += DFS(next);
		}
	}
	return children[node] = sum+1;
}

int main(void)
{
ios::sync_with_stdio(false); cin.tie(NULL);	
	
	cin >> n >> r >> q;
	for(int i=0; i<n-1; i++)
	{
		cin >> v1 >> v2;
		tree[v1].push_back(v2);
		tree[v2].push_back(v1);
	}
	
	DFS(r);
	
	for(int i=0; i<q; i++)
	{
		cin >> u;
		cout << children[u] << "\n";
	}
}
```

트리 DP 문제. 

단순하게 입력에 따라 트리를 만들고, DFS로 완전 탐색 해주면서 각 노드마다 자기 자신 + 자식 노드의 값을 저장해주면 된다.
