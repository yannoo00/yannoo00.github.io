## 문제 정보
- 문제 링크: [백준 9461 파도반 수](https://www.acmicpc.net/problem/9461)
- 난이도: <span style="color:#B5C78A">실버3</span>
- 완료일: 2023년 2월 12일
- 유형: DP

p[1] ~ p[10] 까지의 값이 주어져 있어서 이를 이용해 답의 규칙을 찾아낼 수 있었다.

점화식을 알아내자 코드 작성은 쉬웠다. 

```C++
#include <iostream>

using namespace std;

long long Pado[101];

int main(void)
{
	Pado[1] = 1;
	Pado[2] = 1;
	Pado[3] = 1;
	Pado[4] = 2;
	Pado[5] = 2;
	for(int i= 6; i<101; i++)
	{
		Pado[i] = Pado[i-5]+Pado[i-1];
	}
	
	int n=0;
	
	int k=0;
	cin >> k;
	
	for(int i=0; i<k; i++)
	{
		cin >> n;

		cout << Pado[n]<<"\n";		
	}
	
}
```
