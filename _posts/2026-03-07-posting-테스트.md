### 블로그 포스팅 테스트


그동안은 깃허브 블로그에 포스팅 하기 위해서 vscode를 켜고 .md 파일을 생성하여 글을 작성했다.  

md파일을 생성하고 글 작성 이후에는 commit&push를 해줘야 올라가기 때문에, 
글 하나 쓰려고 해도 코드 편집기를 열어서 이것 저것 해줄게 많아 귀찮았다.  

편하게 포스팅 할 수 있는 방법을 찾다가 notion에 github API 붙이기, 혹은 다양한 github API를 활용해서 직접 편집기를 블로그 내에 구현하기 등의 방법을 찾았고  
이참에 claude code를 활용하여 더 과감하게 블로그를 편집해보기로 했다.   

<img width="1367" height="815" alt="Image" src="https://github.com/user-attachments/assets/d4b93bf7-820d-41ab-a188-3d87b3d692db" />   

우선 글 편집 및 새 글 작성까지 구현해둔 상태이다.  
github action을 통해 배포되기에 정적 사이트의 배포만 가능한데, Github API를 활용하면 웬만한 블로그 기능은 모두 간단하게 구현할 수 있다.
