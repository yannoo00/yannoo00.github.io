property는 클래스 내부 필드에 대한 getter, setter의 구현을 간단하게 돕는다.  
함수이지만 마치 변수처럼 다룰 수 있다.  

```C#
private int hp;
public int Hp
{
    get { return hp; }
    set 
    {
        if (value < 0) hp = 0; // 체력이 음수가 되는 것을 방지
        else hp = value;
    }
}
```  
변수처럼 사용하면서도 함수처럼 내부에 제약 등 구체적인 사항을 구현할 수 있다.
